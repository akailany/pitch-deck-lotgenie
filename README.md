# LotGenie Investor Deck

Static HTML slide deck for LotGenie investor presentations.

## View locally

Open `index.html` in any browser, or serve the folder:

```bash
python3 -m http.server 8766
```

Then visit `http://localhost:8766`.

## Navigate

- Arrow keys / space: next slide
- Shift+arrow / Page Up: previous slide
- Home / End: first / last slide

## Structure

```
investor-deck/
├── index.html          # Slide deck
└── assets/

    ├── small-lot.jpg
    ├── logos/          # Competitor logos
    └── ui/             # Product screenshots
```
